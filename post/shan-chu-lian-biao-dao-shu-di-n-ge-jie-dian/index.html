删除链表倒数第N个节点

· 2024-10-27 ·

题目

Leetcode 26:

给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

题解

  • 转化为删除第N个节点

    class Solution {
public:
    ListNode* removeNthFromEnd(ListNode* head, int n) {
        int length = 0;
        ListNode* curr;
        ListNode* dummy = new ListNode();
        dummy->next = head;
        curr = head;
        while(curr != nullptr) {
            curr = curr->next;
            length ++;
        }
        int right_pos = length - n;
        curr = dummy;
        while(right_pos --) {
            curr = curr->next;
        }
        curr->next = curr->next->next;
        return dummy->next;
    }
};

    通过循环得到链表的长度，将倒数的问题转化为正数的问题。通过增加虚拟头节点统一节点操作，正常写删除节点的代码即可。
